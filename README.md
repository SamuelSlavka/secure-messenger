# Web app with authentication
using SSL and nginx revese proxy

#### Dependencies:
  docker-compose <br>
  docker

#### Setup:
  in file **docker-compose.yml** replace **--staging** with **--force-renew** <br>
    
    ./ssl_renew.sh     
  
#### Starting server:
    docker-compose up
  
 #### Execution:
